# saas-approval-system

Sistema backend para gestionar flujos de aprobación configurables
(encendido/apagado), orientado a productos tipo SaaS.

## Features (MVP)
- Gestión de usuarios
- Roles (Admin / User)
- Flujos de aprobación configurables
- Aprobación manual o automática
- API REST

## Tech stack (planned)
- Backend: NestJS
- DB: PostgreSQL
- Auth: JWT
- Docker
